# 03 — Funcionalidades opcionais

O setup pergunta quais blocos habilitar:

- **browser** — automação de navegador real
- **cron** — lembretes, rotinas e jobs agendados
- **gateway** — editar/reiniciar/validar o gateway por tool
- **message** — envio entre sessões/canais
- **tts** — gerar áudio a partir de texto
- **nodes/canvas** — dispositivos pareados e telas interativas
- **memory-wiki** — wiki persistente/Obsidian-friendly
- **comfy** — workflows ComfyUI para imagem/vídeo/áudio
- **web_search/x_search** — busca web e X/Twitter
- **code_execution** — Python remoto/sandbox para análise rápida

Recomendação inicial segura: `browser, cron, gateway, message, tts, web_search, code_execution`.
